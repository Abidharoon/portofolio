Compte rendu pédagogique : Créer un portfolio avec DeepSeek ou n’importe quelle intelligence Artificielle. Introduction Aujourd’hui, il est indispensable pour un étudiant en informatique ou dans le numérique d’avoir un portfolio en ligne. Ce site personnel permet de montrer son CV, ses compétences, ses projets et de donner une image professionnelle aux recruteurs. Grâce à l’IA, et plus particulièrement avec DeepSeek, il est possible de générer rapidement un portfolio complet sans être un expert en programmation. Ce compte rendu explique, pas à pas, la démarche pour créer son portfolio. 1. Préparer les ressources Avant de commencer, il faut réunir tous les fichiers nécessaires : Un CV à jour au format PDF vous pouvez utiliser CV designer ou le site que vous préférez pour créer votre CV. Une photo de profil professionnelle, renommée simplement pour pouvoir la retrouver plus facilement. Un dossier local sur l’ordinateur pour ranger tous les fichiers, votre CV et votre photo doivent être placer dans le même fichier que vous avez créé et renommer. 2. Générer le code du portfolio avec DeepSeek 1. Se connecter à DeepSeek (ou une autre IA comme ChatGPT ou autre. 2. Vous allez envoyer votre CV a Deepseek ou une autre IA celle que vous préférez et vous allez mettre le texte en bas pour créer votre portefolio et après il faut modifier vos informations car L’IA mettras des informations au hasard. 3. ( Je joins mon CV ici ; utilisez-le comme référence complète. J'ai une photo enregistrée localement et un fichier CV dans mon dossier, et je vous demande de créer pour moi un site web de portfolio personnel de haute qualité et remarquable, utilisant HTML, CSS, JavaScript et Bootstrap 5. Le site doit être entièrement autonome dans un seul fichier HTML (avec tous les CSS et JS intégrés) et inclure des sections essentielles telles que Accueil, À propos de moi, Compétences (présentées visuellement), Projets (avec images et fenêtres contextuelles pour plus de détails), Expérience, Contact, des liens vers les réseaux sociaux et un bouton de téléchargement de CV qui déclenche un téléchargement immédiat sans être bloqué par le navigateur. Utilisez des images de substitution de haute qualité provenant d'Unsplash pour les projets, assurez-vous que tout est bien formulé dans un français correct et que le design général soit professionnel, soigné et visuellement attrayant pour me démarquer) 4. Une fois vous avez lancer la génération ca prendras quelque minute pour pouvoir générer votre code une fois le code a été créer vous allez copier le code pour pouvoir le mettre dans un fichier (vous pouvez utiliser document texte ou ce que vous voulez) et après avoir enregistrer votre fichier vous allez modifier les extensions de fichier pour pouvoir changer le code et avoir votre portefolio. Les extensions de votre fichier seront .odt où. Word vous allez renommer votre fichier en Index.html et normalement cela doit être bon. Dans le cas contraire vous allez faire afficher les extensions de fichier et les changer après. 5. Une fois cela est fait normalement vous allez avoir une interface avec une photo et tout vos informations avec un lien ou un bouton qui permettras de télécharger votre CV. 6. Vous pouvez aussi faire les changements des information sur votre portfolio il faudra modifier le code que vous avez dans votre fichier index.html vous pouvez le faire vous-même mais ça va être plus simple et plus facile si vous demander a DEEPSEEK ou a CHATgpt ce que vous voulez modifier. 5. Héberger le portfolio en ligne Jusqu’ici, le site est visible uniquement sur votre ordinateur. Pour qu’il soit accessible par un recruteur, il faut l’héberger. • Pour héberger votre site en ligne il faudra créer un site sur Github et Netlify • Vous allez envoyer votre CV et votre index.html sur Github. • Une fois cela est fait vous allez créer un compte sur Netlify et vous allez connecter votre Github à Netlify pour pouvoir déployer votre site en ligne. • Connecter Netlify à GitHub pour déployer le site. • Netlify vous donneras une adresse que vous pouvez partager et qui vous permettras d’accéder votre site depuis un autre ordinateur en ligne et qui permettras a d’autres personne aussi de voir votre site et votre CV en ligne avec le lien partager par vous. Conclusion Avec DeepSeek ou n’importe quelle IA, il est possible de créer rapidement un portfolio professionnel même sans connaissances techniques poussées. Les étapes clés sont : 1. Préparer son CV et sa photo. 2. Générer le code avec un prompt précis. 3. Enregistrer et ouvrir le fichier index.html. 4. Personnaliser les informations et les liens. 5. Héberger le site en ligne avec GitHub et Netlify. Ce tutoriel montre qu’il est aujourd’hui possible, grâce à l’IA, de gagner du temps et de disposer d’un site personnel moderne et efficace, essentiel pour se démarquer lors de recherches de stages ou d’emplois.
